# Logo/Mascota Generator Prompt

Copy and paste this prompt into your favorite AI image generator (Midjourney, DALL-E, Stable Diffusion, Leonardo AI, etc.).

---

## Primary Prompt: DevPixelForge Mascot Logo

```
Design a modern mascot logo for "DevPixelForge" - a high-performance media processing engine written in Rust with Go bindings.

The mascot should be a friendly pixel art robot or creature that represents:
- Image, video, and audio processing
- High-performance computing (Rust)
- Digital transformation and media manipulation
- Developer-friendly tooling

Style requirements:
- Flat design, modern, minimalist
- Color palette: deep blue (#1E3A5F), electric cyan (#00D4FF), vibrant purple (#8B5CF6), white accents
- Vector-friendly, clean lines
- Square format (1:1) for avatar/icon use
- Professional tech company aesthetic

Optional elements to incorporate:
- Pixel/grid patterns (evoking "pixel" in the name)
- Abstract shapes representing media files (video frame, audio wave, image)
- A transformation/transmutation symbol
- Gear or processing iconography

The mascot should look trustworthy, capable, and approachable - like a helpful digital artisan.

---

## Alternative Prompt: Logo Mark Only (No Mascot)

```
Design a minimal logo mark for "DevPixelForge" - a Rust media processing library with Go integration.

Design elements:
- Abstract "DP" monogram OR geometric symbol representing media processing
- Transformation motif (before → after, input → output)
- Pixel/grid aesthetic

Style:
- Flat, modern, clean
- Colors: deep blue (#1E3A5F), cyan (#00D4FF), purple (#8B5CF6)
- Monochrome version in white for dark backgrounds
- Scalable vector format

Think: professional developer tool, reliable infrastructure software.

## Mascot Character Ideas

### Option 1: The Pixel Wizard
A small robot wizard holding a magic pixel wand, transforming raw media into optimized output. Friendly face, cape made of flowing code.

### Option 2: The Forge Spirit
A fire elemental made of molten pixels, hammering media into perfect form. Represents the "forge" in DevPixelForge - crafting and transforming.

### Option 3: The Digital Fox
Clever fox made of pixel blocks, representing intelligence and speed (Rust is fast). Wearing developer goggles, holding a video frame or audio wave.

### Option 4: The Media Alchemist
A scientist/chemist mixing RGB colors and audio waveforms in a cauldron, transforming inputs into gold (optimized output).

### Option 5: The Pixel Bee
Hard-working bee made of pixel squares, representing parallel processing (Rayon parallelism). Multiple wings showing threads/cores.

---

## Usage Guidelines

1. **Primary use**: GitHub repo README, website header
2. **Secondary use**: Social media avatars, favicon
3. **Colors**: Keep original palette, can adapt to monochrome for printing
4. **Format**: Request PNG + SVG versions
5. **Variations**: Request horizontal (wide) and square versions
